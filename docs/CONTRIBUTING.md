# Contributing to MARL-SmartGrid-Pricing

Thank you for your interest in contributing to the **MARL-SmartGrid-Pricing** project!
Your contributions help improve and expand this research framework.


## How You Can Contribute
We welcome all types of contributions, including:
- Adding new features or agents
- Fixing bugs or improving performance
- Enhancing documentation or examples
- Refactoring or optimizing existing modules
- Writing or improving tests
- Suggesting improvements for training or reward mechanisms

## Getting Started

1. **Fork the Repository**
   Click the **Fork** button on GitHub to create your copy of the repo.

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/your-username/marl-smartgrid-pricing.git
   cd marl-smartgrid-pricing
   ```

3. **Create a Branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Changes**
   - Implement and test your changes locally
   - Follow **PEP 8** coding standards
   - Keep commits small and meaningful

5. **Commit and Push**
   ```bash
   git add .
   git commit -m "Brief but descriptive message"
   git push origin feature/your-feature-name
   ```

6. **Open a Pull Request**
   - Go to your fork and click **Compare & pull request**
   - Provide a clear title and description
   - Link related issues if applicable

## Contribution Guidelines
- Keep code modular, readable, and documented
- Write unit tests for new or modified functionality
- Do not commit large files, datasets, or generated outputs
- Regularly sync your fork:
  ```bash
  git remote add upstream https://github.com/yourusername/marl-smartgrid-pricing.git
  git fetch upstream
  git merge upstream/main
  ```


## Need Help?
If you have any questions, open an **issue** or start a **discussion** in the repository.
We appreciate your effort and contribution to advancing this MARL research.
